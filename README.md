PROJECT NAME:  AWS Serverless Web Application

This is a simple serverless web application built using AWS services.It includes a static frontend hosted on Amazon S3 and a backend powered by API Gateway, AWS Lambda, and Amazon DynamoDB.
The project demonstrates core cloud concepts such as serverless architecture, API integration, backend automation, database operations, and secure, scalable design.

1.PROJECT OVERVIEW:

This project demonstrates how to build a full serverless application using:
1.S3 – Frontend hosting
2.API Gateway – Exposes API endpoint
3.AWS Lambda – Backend logic
4.DynamoDB – Stores page view count
The frontend is built using HTML, CSS, JavaScript, and connects to AWS services through an API.

2.Features

Serverless architecture (no servers to manage)
Frontend hosted on Amazon S3
API built using API Gateway
Backend logic implemented with AWS Lambda (Python)
Visitor count stored and updated in DynamoDB
Interactive UI with JavaScript
